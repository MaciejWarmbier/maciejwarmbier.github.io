Tower Defense
Tower Defense Game with pathfinding mechanic in which I mostly played with number of different objects and classes. Enemies are created with object pool adn now is where interesting part comes. As enemies can go through previously made path or find it by their own.  Pathfinding is based on BFS algorithm. There is also simple Bank Mechanic which rewards player for killing and enemy and penalize if any reaches the end. Whole project finally starts to utilize Object Oriented programming which I find myself most comfortable with. 

https://github.com/MaciejWarmbier/Tower-Defense.git

Space War
A simple Rail Shooter game created in Unity. Its main focus is creating a fast and chatic experience for the player while he needs to destroy wave by wave of enemies. Controlling so many actors was my experiment here. So for both player trail and enemy movement I used Timeline tool. Player movement is controled by some simple movement Inputs and it works with both keyboard and xbox controller. Adding a lot of particle effects with dynamic deletion of them was also a new experience for me. 

https://github.com/MaciejWarmbier/Space-war.git

Starship
A 2.5D simple obstacle game with basic physics. Figuring out player movement was most tricky part for it to feel smooth and good to control. Additionally I played with basic object movement and physics, like some falling debree when player is close enough. For now game has only 3 levels but it might change in the future. 

https://github.com/MaciejWarmbier/Starship-game.git

Sables
A little Python project I did during my studies. It is multiplayer game where player tries to kill each other and step into each other territory. Mechanics are pretty simple as plyers have 3 height to choose to attack from. Game uses Pyhton Pygame and Socket extensions and can be played on local network usind IPv4 protocol. 

https://github.com/MaciejWarmbier/Sables-game-python-sockets.git

Obstacle Course
Little obstacle course game with 2.5D perspective. In this I was just playing with simple ideas such as possible movement and different obstacles based on different mechanics.

https://github.com/MaciejWarmbier/Simple-obstacle-course-unity.git
